Fetches images from Pexels based on a specified topic, directly within your WordPress site.

== Description ==

Pexels Image Fetcher allows you to enrich your website with beautiful, relevant photos from Pexels. With this plugin, you can dynamically add images to your site based on specified topics, directly integrating with the Pexels API. Enhance your blog posts, pages, or any part of your site with visually appealing content.

== Features ==

- Easy to use with a simple shortcode.
- Dynamically fetch images from Pexels based on the topic you choose.
- Control the number of images displayed.
- Customize the display with your own CSS.
- Secure and optimized for performance.

== Installation ==

1. Upload the `pexels-image-fetcher` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Settings' -> 'Pexels Image Fetcher' to configure the plugin.
4. Enter your Pexels API key in the provided field and save the changes.

== Usage ==

Simply use the shortcode `[pexels_images topic="nature" num_images="10"]` in your posts or pages. Replace `nature` with your desired topic and `10` with the number of images you want to display.

== Frequently Asked Questions ==

= Where do I get a Pexels API key? =

You can obtain a Pexels API key by signing up at [Pexels Developers](https://www.pexels.com/api/new/).

= Is there a limit to the number of images I can fetch? =

The number of images depends on your Pexels API plan. Please refer to Pexels' API documentation for rate limits and usage guidelines.
